Subtema 2.1

Variables y Dataframes en R

En R, los datos se almacenan en objetos. El operador <- crea objetos a partir de valores o funciones.

i
Tipos de variables en R

String (texto): valores entre comillas "así" — R los trata como texto.
Numérico: valores sin comillas 8 6.5 42 — R los trata como números.

Crear variables con c()

R · c() Crear vectores 
# Variable de texto (string)
nombresEstudiantes <- c("Ana", "Luis", "Sofia", "Carlos", "Maria")

# Variable numérica — sin comillas
notasEstudiantes   <- c(8, 6, 9, 7, 5)
edadEstudiantes    <- c(20, 22, 21, 23, 20)

Combinar en un Dataframe

Un dataframe es como una hoja de cálculo: organiza variables en columnas y observaciones en filas.

R · data.frame() Crear dataframe 
salon <- data.frame(
  Nombre = nombresEstudiantes,
  Nota   = notasEstudiantes,
  Edad   = edadEstudiantes
)

salon              # ver dataframe
names(salon)      # listar columnas
salon$horasEstudio <- c(12, 6, 15, 9, 4)  # agregar columna

Subtema 2.2

Limpieza y transformación con tidyverse

El paquete tidyverse agrupa herramientas para manipular datos de forma legible y encadenada. Su función central es el operador %>% (pipe), que pasa el resultado de una operación a la siguiente.

✓
¿Por qué tidyverse?

En investigación real, el 70-80% del tiempo de análisis se dedica a limpiar y preparar datos. tidyverse convierte operaciones complejas en código legible y reproducible.

Funciones principales de dplyr

Función¿Qué hace?Ejemplo
filter()Filtra filas según condiciónfilter(df, nota >= 6)
select()Elige columnas específicasselect(df, nombre, nota)
mutate()Crea o modifica columnasmutate(df, nivel=case_when(...))
arrange()Ordena filasarrange(df, desc(nota))
group_by()Agrupa por categoríagroup_by(df, genero)
summarise()Calcula resúmenes por gruposummarise(df, media=mean(nota))
rename()Renombra columnasrename(df, edad_anios=edad)
replace_na()Reemplaza valores NAreplace_na(list(nota=0))

El operador pipe %>%

R · pipe Encadenar operaciones 
# Sin pipe — difícil de leer
arrange(select(filter(encuesta, nota > 6), nombre, nota), desc(nota))

# Con pipe — se lee de arriba a abajo
encuesta %>%
  filter(nota > 6) %>%
  select(nombre, nota) %>%
  arrange(desc(nota))

Tratamiento de valores faltantes (NA)

Según Herrera (2020), antes de cualquier prueba estadística deben tratarse los missing values y valores atípicos. tidyverse facilita ambas tareas.

R · NA Detectar y tratar NA 
# Detectar NA por columna
colSums(is.na(encuesta))

# Filtrar filas sin NA en una columna específica
encuesta %>% filter(!is.na(nota))

# Reemplazar NA con la mediana
mediana_h <- median(encuesta$horas, na.rm = TRUE)
encuesta %>%
  mutate(horas = replace_na(horas, mediana_h))

★
Contexto

Parte 0 — Instalación y dataset

0

Script completo — Práctica 2.2

# PRÁCTICA 2.2 — Limpieza con tidyverse
library(tidyverse)

# Dataset
set.seed(42)
encuesta <- tibble(id=1:200,edad=c(sample(22:55,190,replace=T),rep(NA,10)),
              genero=sample(c("M","F",NA),200,replace=T,prob=c(.45,.45,.1)),
              nota=c(round(runif(180,4,10),1),rep(NA,20)),
              horas_sem=c(sample(4:30,185,replace=T),rep(NA,15)),
              departamento=sample(c("Cuenca","Quito","Guayaquil"),200,replace=T))

# P1 Exploración
glimpse(encuesta); colSums(is.na(encuesta))

# P2 Filtrar y seleccionar
enc_limpia <- encuesta %>%
  filter(nota >= 6, !is.na(edad)) %>%
  select(id, genero, nota, horas_sem) %>%
  arrange(desc(nota))

# P3 Transformar
encuesta <- encuesta %>%
  mutate(
    nivel_nota = case_when(nota >= 8 ~ "Alto", nota >= 6 ~ "Medio", TRUE ~ "Bajo"),
    horas_sem  = replace_na(horas_sem, median(horas_sem, na.rm=T))
  ) %>%
  rename(horas_semanales = horas_sem)

# P4 Agrupar
encuesta %>%
  group_by(nivel_nota) %>%
  summarise(n=n(), nota_media=mean(nota,na.rm=T), horas_media=mean(horas_semanales,na.rm=T)) %>%
  arrange(desc(nota_media))

encuesta %>% count(genero, nivel_nota) %>% arrange(genero, desc(n))
