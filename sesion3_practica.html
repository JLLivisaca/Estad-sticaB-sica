colSums(is.na(estudiantes))

Reflexión: Si nota_final falta solo en quienes no aprobaron, el mecanismo es MNAR — imputar con la media sería un error porque sesgaría la distribución hacia arriba.
3
Detecta outliers en nota_final con la regla IQR (Tukey) y con Z-score. ¿Coinciden los resultados? ¿Eliminarías esos valores?
IQR(), boxplot.stats(), scale()
▾ 
# Método IQR (Tukey)
notas <- na.omit(estudiantes$nota_final)
Q1 <- quantile(notas, 0.25)
Q3 <- quantile(notas, 0.75)
iqr <- IQR(notas)
lim_inf <- Q1 - 1.5 * iqr
lim_sup <- Q3 + 1.5 * iqr
outliers_iqr <- notas[notas < lim_inf | notas > lim_sup]
cat("Outliers IQR:", outliers_iqr, "\n")

# Método Z-score (|z| > 3)
z <- as.numeric(scale(notas))
outliers_z <- notas[abs(z) > 3]
cat("Outliers Z-score:", outliers_z, "\n")

# Boxplot visual
boxplot(notas, main = "Distribución nota_final",
        col = "#4A90E2", ylab = "Nota")
Decisión documentada: Si el outlier es un error de captura (nota = 15 en escala de 0–10), se elimina. Si es un estudiante con rendimiento genuinamente excepcional, se conserva y se menciona en el reporte.
4
Imputa los valores faltantes en nota_final usando la mediana (más robusta que la media para datos asimétricos). Verifica que no queden NAs.
median(na.rm=TRUE), is.na()
▾ 
# Imputación con mediana
mediana_nota <- median(estudiantes$nota_final, na.rm = TRUE)
estudiantes$nota_final[is.na(estudiantes$nota_final)] <- mediana_nota

# Verificación
sum(is.na(estudiantes$nota_final))  # debe dar 0
cat("Mediana usada para imputación:", mediana_nota, "\n")
¿Por qué mediana y no media? Si la distribución de notas es asimétrica (lo comprobarás en la Parte 3), la media está sesgada por los outliers. La mediana es más representativa como valor de imputación.
Parte 2 3.1 — Tendencia central

Media, mediana y moda

5
Calcula media, mediana y moda de nota_final. ¿Difieren notablemente? ¿Qué medida reportarías en un paper y por qué?
mean(), median(), which.max(table())
▾ 
# Las tres medidas
media   <- mean(estudiantes$nota_final)
mediana <- median(estudiantes$nota_final)
moda    <- as.numeric(names(which.max(table(
             round(estudiantes$nota_final, 1)))))

cat(sprintf("Media:   %.2f\nMediana: %.2f\nModa:    %.1f\n",
          media, mediana, moda))

# Media recortada (5%) — robusta ante outliers extremos
media_trim <- mean(estudiantes$nota_final, trim = 0.05)
cat("Media cortada (5%):", round(media_trim, 2), "\n")
Regla APA §4.19: Si |media − mediana| > 0.5 puntos en esta escala, reporta ambas. Si además encontraste outliers en la Parte 1, prioriza la mediana como medida principal.

Parte 3 3.2 — Dispersión

Varianza, SD, asimetría y curtosis

7
Calcula varianza, desviación estándar, asimetría y curtosis de nota_final. Interpreta cada resultado e indica si la distribución es simétrica.
var(), sd(), library(e1071)
▾ 
library(e1071)

varianza <- var(estudiantes$nota_final)
desv_est <- sd(estudiantes$nota_final)
asimetria <- skewness(estudiantes$nota_final)
curtosis  <- kurtosis(estudiantes$nota_final)

cat(sprintf("Varianza:  %.4f\nDesv. Est: %.4f\nAsimetría: %.4f\nCurtosis:  %.4f\n",
          varianza, desv_est, asimetria, curtosis))

# Interpretación automática
if(abs(asimetria) > 1) {
  cat("→ Distribución NOTABLEMENTE asimétrica. Reportar mediana.\n")
} else {
  cat("→ Distribución aproximadamente simétrica. Media es adecuada.\n")
}
EstadísticoValor esperadoInterpretación
Asimetría (γ₁)γ₁ ≈ 0Simétrica
γ₁ > 1Cola derecha pronunciada — usar mediana
Curtosis (γ₂)γ₂ = 3Normal (mesocúrtica)
γ₂ > 3Leptocúrtica — probable presencia de outliers
8
Compara la desviación estándar de nota_final entre carreras. ¿En qué carrera son las notas más homogéneas?
▾ 
# SD por carrera
sd_carrera <- tapply(estudiantes$nota_final,
                     estudiantes$carrera, sd)
round(sort(sd_carrera), 3)

# Coeficiente de variación (CV) = SD/Media — comparación relativa
cv_carrera <- tapply(estudiantes$nota_final, estudiantes$carrera,
                     function(x) sd(x)/mean(x) * 100)
round(sort(cv_carrera), 1)
CV: El coeficiente de variación permite comparar dispersión entre grupos con medias distintas. CV < 15% = datos homogéneos; CV > 30% = datos heterogéneos.

Construye una tabla de frecuencias para nota_final (variable continua) usando la regla de Sturges para determinar el número de clases. Luego crea el histograma.

n_clases <- nclass.Sturges(estudiantes$nota_final)

           F_n = round(as.numeric(Fn), 3))

# Histograma
hist(estudiantes$nota_final,
     breaks = n_clases,
     col    = "#4A90E2",
     main   = "Distribución de notas finales",
     xlab   = "Nota final",
     ylab   = "Frecuencia")

# Ojiva (frecuencia acumulada)
plot(ecdf(estudiantes$nota_final),

Regla de Sturges: k ≈ 1 + 3.322 · log₁₀(n). Para n = 80 → k ≈ 7 clases. Puedes ajustar manualmente si los intervalos no tienen sentido práctico.

★
Bonus: Detecta outliers multivariados en nota_final + horas_estudio usando la distancia de Mahalanobis. ¿Hay estudiantes con un patrón de notas/horas inusual?
Nivel avanzado — mahalanobis(), chi-cuadrado
▾ 
# Outliers multivariados — Distancia de Mahalanobis
vars_num <- estudiantes[, c("nota_final", "horas_estudio")]

mah <- mahalanobis(vars_num,
                   colMeans(vars_num),
                   cov(vars_num))

# Umbral: chi² con p = 0.001 y 2 grados de libertad
umbral <- qchisq(0.999, df = 2)
outliers_mv <- which(mah > umbral)

cat("Outliers multivariados (índices):", outliers_mv, "\n")
estudiantes[outliers_mv, c("nota_final", "horas_estudio", "carrera")]

# Gráfico: scatter con outliers marcados
plot(vars_num, pch = 16, col = "#4A90E2",
     main = "Outliers multivariados (Mahalanobis)")
points(vars_num[outliers_mv,], pch = 4, col = "#C0392B", cex = 2, lwd = 2)
legend("topright", legend = c("Normal", "Outlier Mah."),
       pch = c(16, 4), col = c("#4A90E2", "#C0392B"))

# Datos de rendimiento académico (n = 80, 6 variables)

n <- 80

  id           = 1:n,
  nota_final   = round(c(rnorm(74, mean = 7.1, sd = 1.4),
                           c(0.8, 0.9, 10.5, 10.8, 11.0, 10.9)), 1),

                           "Derecho", "Administración"),

estudiantes$nota_final <- pmin(pmax(estudiantes$nota_final, 0), 10)
