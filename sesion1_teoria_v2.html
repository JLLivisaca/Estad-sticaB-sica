Subtema 1.1

La Estadística como herramienta de investigación científica

La estadística no es solo cálculo: es el lenguaje con el que la ciencia toma decisiones bajo incertidumbre. Como lo plantea Devore (2008), la estadística "proporciona métodos de organizar y resumir datos y de sacar conclusiones basadas en la información contenida en los datos."

"El objetivo fundamental de la Estadística es analizar datos y transformarlos en información útil para tomar decisiones."

— Rodríguez Ojeda, ESPOL (2007)

Conceptos fundamentales

Concepto Definición
EstadísticaCiencia inductiva que permite inferir características de un conjunto a partir de un subconjunto (muestra)
PoblaciónConjunto total de individuos u objetos con una característica de interés
MuestraSubconjunto representativo seleccionado de la población
ParámetroCaracterística numérica de la población, generalmente desconocida
EstadísticoValor calculado a partir de la muestra para estimar un parámetro
VariableCaracterística observable que toma distintos valores entre sujetos

Tipos de variables

TipoSubtipoEjemplo en investigación
CategóricaNominal / OrdinalÁrea disciplinar, nivel de satisfacción
Numérica discretaPor conteoNúmero de publicaciones, número de citas
Numérica continuaPor mediciónTiempo de respuesta, temperatura, peso

Las dos ramas fundamentales

Rama¿Qué hace?Ejemplo en investigación
Estadística descriptivaRecopila, organiza y presenta datosTablas de frecuencia, gráficos, medias
Estadística inferencialExtiende conclusiones de la muestra a la poblaciónPruebas de hipótesis, intervalos de confianza
i
Reflexión para investigadores

Cuando se publica un paper, los métodos estadísticos usados determinan si las conclusiones son válidas y reproducibles. Un análisis incorrecto puede invalidar todo el estudio, independientemente de la calidad del diseño experimental.

Subtema 1.2

Significancia estadística vs. Relevancia práctica

Esta es una de las distinciones más críticas y frecuentemente ignoradas en la investigación académica. Confundir ambos conceptos puede llevar a publicar hallazgos matemáticamente correctos pero científicamente irrelevantes.

El valor p — definición y límites

"El valor P es el nivel de significación más pequeño al cual H₀ sería rechazada con un conjunto de datos dado."

— Devore, J.L. (2008), Cap. 8

El valor p mide la probabilidad de observar los datos obtenidos si la hipótesis nula fuera verdadera. No mide la magnitud del efecto, ni la importancia práctica del hallazgo.

!
El problema del tamaño de muestra grande

Devore (2008) advierte: un valor p pequeño "puede ser el resultado de un tamaño de muestra grande en combinación con un alejamiento de H₀ que tiene poca significación práctica." Con muestras grandes, diferencias triviales se vuelven estadísticamente significativas.

Ilustración: misma diferencia, distinto valor p

Se prueba H₀: μ = 100 con valor real μ = 101 (diferencia de solo 1 unidad en todos los casos):

Tamaño de muestra (n)Valor pConclusión
250.3085No significativo
4000.0228Significativo (p < 0.05)
2,5000.0000003Muy significativo
10,0007.69 × 10⁻²⁴Extremadamente significativo

La solución: reportar ambas medidas

MedidaPregunta que respondeIndicadores
Valor p¿Es el resultado producto del azar?p < 0.05, p < 0.01
Tamaño del efecto¿Qué tan grande es la diferencia real?Cohen's d, η², r
✓
Regla de oro para publicación científica

Un resultado estadísticamente significativo sin reporte del tamaño del efecto es información incompleta. Las revistas Q1 cada vez más exigen ambos valores como requisito de publicación.

¿Por qué este artículo?

El contexto es cercano a usted como participante — son docentes universitarios. Usa estadística descriptiva básica (medias, SD, frecuencias), tiene datos claros y reproducibles, y es un modelo real de reporte en revista Q1. Además conecta directamente con el Subtema 1.2: los autores reportan valor p pero ¿reportan tamaño del efecto?

Paso 1 Simular el dataset (n = 240, parámetros del artículo) 
set.seed(2026)  # semilla para reproducibilidad
n <- 240

datos <- tibble(
  id          = 1:n,
  edad        = round(rnorm(n, mean = 38.6, sd = 8.2)),
  sexo        = sample(c("Femenino", "Masculino"),
                        size = n, replace = TRUE,
                        prob = c(0.583, 0.417)),
  rango       = sample(c("Asistente", "Asociado", "Titular"),
                        size = n, replace = TRUE,
                        prob = c(0.45, 0.35, 0.20)),
  anios_exp         = round(rnorm(n, mean = 10.3, sd = 7.1)) |> pmax(0),
  mbi_agotamiento   = round(rnorm(n, mean = 3.12, sd = 1.45), 1) |> pmin(6) |> pmax(0),
  mbi_despers       = round(rnorm(n, mean = 1.89, sd = 1.23), 1) |> pmin(6) |> pmax(0),
  mbi_realizacion   = round(rnorm(n, mean = 3.67, sd = 1.18), 1) |> pmin(6) |> pmax(0),
  wpai_perdida      = round(rnorm(n, mean = 28.4, sd = 18.7), 1) |> pmin(100) |> pmax(0)
)

glimpse(datos)

Paso 4 Visualización con ggplot2 — 4 gráficos 
# ── Gráfico 1: Histograma de edad ──────────────────────────
ggplot(datos, aes(x = edad)) +
  geom_histogram(binwidth = 5, fill = "#1d4ed8", color = "white", alpha = 0.85) +
  geom_vline(xintercept = mean(datos$edad),
             color = "#b45309", linewidth = 1, linetype = "dashed") +
  labs(title = "Distribución de edad — Docentes universitarios",
       subtitle = "n = 240 | Amer et al. (2022)",
       x = "Edad (años)", y = "Frecuencia") +
  theme_minimal()

# ── Gráfico 2: Boxplot agotamiento por rango ────────────────
ggplot(datos, aes(x = rango, y = mbi_agotamiento, fill = rango)) +
  geom_boxplot(alpha = 0.7) +
  geom_jitter(width = 0.15, alpha = 0.25, size = 1) +
  labs(title = "Agotamiento emocional por rango académico",
       x = "Rango", y = "MBI Agotamiento (0–6)") +
  theme_minimal() + theme(legend.position = "none")

# ── Gráfico 3: Barras nivel burnout por sexo ────────────────
datos |>
  mutate(nivel_burnout = case_when(
    mbi_agotamiento >= 4.0 ~ "Alto",
    mbi_agotamiento >= 2.0 ~ "Moderado",
    TRUE                   ~ "Bajo"
  )) |>
  count(sexo, nivel_burnout) |>
  group_by(sexo) |>
  mutate(pct = n / sum(n) * 100) |>
  ggplot(aes(x = sexo, y = pct, fill = nivel_burnout)) +
  geom_col(position = "dodge", alpha = 0.85) +
  labs(title = "Nivel de burnout por sexo",
       x = "Sexo", y = "Porcentaje (%)") +
  theme_minimal()

# ── Gráfico 4: Dispersión agotamiento vs productividad ──────
ggplot(datos, aes(x = mbi_agotamiento, y = wpai_perdida)) +
  geom_point(alpha = 0.5, color = "#1d4ed8") +
  geom_smooth(method = "lm", se = TRUE, color = "#c0392b") +

Paso 5 Reflexión: valor p vs. tamaño del efecto (Cohen's d) 
# Prueba t: Agotamiento — Asistente vs. Titular
grupo_a <- datos |> filter(rango == "Asistente") |> pull(mbi_agotamiento)
grupo_t <- datos |> filter(rango == "Titular")   |> pull(mbi_agotamiento)

prueba_t <- t.test(grupo_a, grupo_t)
print(prueba_t)

# Tamaño del efecto: Cohen's d
pooled_sd <- sqrt((sd(grupo_a)^2 + sd(grupo_t)^2) / 2)
cohen_d   <- (mean(grupo_a) - mean(grupo_t)) / pooled_sd

cat("Valor p     =", round(prueba_t$p.value, 4), "\n")
cat("Cohen's d   =", round(cohen_d, 3), "\n")
cat("Interpretación: |d| < 0.2 negligible · ~0.5 mediano · ≥ 0.8 grande\n")

# Guardar dataset para Sesión 2
write_csv(datos, "burnout_academico_2026.csv")
cat("✓ Dataset guardado: burnout_academico_2026.csv\n")
